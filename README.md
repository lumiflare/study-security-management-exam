# 情報セキュリティマネジメント試験 学習プロジェクト

Claude Code をチューターとして活用し、**情報セキュリティマネジメント試験（SG試験）** の合格を目指す学習プロジェクトです。

## 試験概要

| 項目 | 内容 |
|------|------|
| 試験日 | 2026年4月26日（日） |
| 形式 | CBT方式、120分 |
| 科目A | 48問（四肢択一）— セキュリティ中心 |
| 科目B | 12問（多肢選択）— ケーススタディ |
| 合格基準 | 1000点満点中600点以上（IRT方式） |

## プロジェクト構成

```
.
├── CLAUDE.md          # チューター設定（対話ルール・出題分野など）
├── study-plan.md      # 7日間学習計画
├── notes/             # トピック別の概念ノート
│   └── security-basics.md
└── progress/          # 学習進捗の記録
    └── wrong-answers.md
```

## 使い方

Claude Code で以下のスラッシュコマンドを使って学習を進めます。

| コマンド | 説明 |
|----------|------|
| `/explain` | トピックの解説を受ける |
| `/quiz` | 科目A形式の4択クイズに挑戦する |
| `/mock-exam` | 模擬試験を実施する |
| `/review` | 今日の学習内容を振り返る |
| `/progress` | 7日間計画に対する進捗を確認する |
| `/weak-points` | 間違えた問題から弱点を分析する |

## 学習計画（7日間）

1. **Day 1** — セキュリティ基礎（CIA三要素、攻撃手法、マルウェア）
2. **Day 2** — 暗号・認証技術（共通鍵/公開鍵暗号、PKI、多要素認証）
3. **Day 3** — ネットワークセキュリティ（FW、IDS/IPS、VPN、TLS）
4. **Day 4** — セキュリティ管理（ISMS、リスクマネジメント、BCP/DR）
5. **Day 5** — 法規・制度（個人情報保護法、不正アクセス禁止法など）
6. **Day 6** — 科目B対策（ケーススタディ演習）
7. **Day 7** — 総仕上げ（模擬試験・弱点復習）

詳細は [study-plan.md](study-plan.md) を参照してください。
