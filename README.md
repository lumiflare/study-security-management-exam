# 情報セキュリティマネジメント試験 学習プロジェクト

Claude Code をチューターとして活用し、**情報セキュリティマネジメント試験（SG試験）** の合格を目指す学習プロジェクトです。

## 試験概要

| 項目 | 内容 |
|------|------|
| 試験日 | 2026年4月26日（日） |
| 形式 | CBT方式、120分 |
| 科目A | 48問（四肢択一）— セキュリティ中心 |
| 科目B | 12問（多肢選択）— ケーススタディ |
| 採点対象 | 60問中54問（6問は評価用で採点対象外） |
| 合格基準 | 1000点満点中600点以上（IRT方式） |

## 使い方

Claude Code で以下のスラッシュコマンドを使って学習を進めます。

| コマンド | 説明 |
|----------|------|
| `/explain` | トピックの解説を受ける |
| `/quiz` | 科目A形式の4択クイズに挑戦する（Notion に直接出題） |
| `/mock-exam` | 模擬試験を実施する |
| `/review` | 今日の学習内容を振り返る |
| `/progress` | 7日間計画に対する進捗を確認する |
| `/weak-points` | 間違えた問題から弱点を分析する |

> 練習問題は Notion MCP 経由で「情報セキュリティマネジメント試験」ページ配下に書き込まれます。詳細は [CLAUDE.md](CLAUDE.md) を参照。

## 学習計画（7日間）

| Day | 日付 | テーマ |
|-----|------|--------|
| Day 1 | 4/19(日) | セキュリティ基礎（CIA三要素、攻撃手法、マルウェア） |
| Day 2 | 4/20(月) | 暗号・認証技術（共通鍵/公開鍵暗号、PKI、多要素認証） |
| Day 3 | 4/21(火) | ネットワークセキュリティ（FW、IDS/IPS、VPN、TLS） |
| Day 4 | 4/22(水) | セキュリティ管理（ISMS、リスクマネジメント、BCP/DR） |
| Day 5 | 4/23(木) | 法規・制度（個人情報保護法、不正アクセス禁止法など） |
| Day 6 | 4/24(金) | 科目B対策（ケーススタディ演習） |
| Day 7 | 4/25(土) | 総仕上げ（模擬試験・弱点復習） |

詳細は [study-plan.md](study-plan.md) を参照してください。

## 1日の学習サイクル（3〜4時間）

1. **概念学習**（1〜1.5時間）: `/explain` で各トピックを学習
2. **クイズ演習**（1〜1.5時間）: `/quiz` で理解度をチェック
3. **復習**（30分〜1時間）: `/review` で振り返り、弱点を確認

## 学習スタイル

試験直前、7日前から勉強を始めたので、概念整理は1日でざっくり把握し問題を解きながら勉強した。
