情報セキュリティマネジメント試験の科目A形式の4択クイズを出題してください。

## ルール
- 引数が指定されている場合はその分野から出題: $ARGUMENTS
- 引数がない場合はランダムな分野から出題
- 1回につき3問出題する
- 各問題は本番試験に近い形式（四肢択一）で出題する
- ユーザーが回答したら、正誤を判定し、詳しい解説を表示する
- 間違えた問題は `progress/wrong-answers.md` に追記する（ファイルがなければ作成）
- 全問終了後、正答率と弱いポイントをフィードバックする

## 出題形式
```
**問題 N**（分野: ○○）

[問題文]

ア. [選択肢A]
イ. [選択肢B]
ウ. [選択肢C]
エ. [選択肢D]
```

## 解説形式
```
**正解: [ア/イ/ウ/エ]**

[なぜその選択肢が正解なのか、他の選択肢はなぜ不正解なのかの解説]

💡 **試験ポイント**: [この問題に関連して覚えておくべきこと]
```

## 出題可能な分野
- セキュリティ基礎（CIA、脅威、脆弱性）
- 攻撃手法（マルウェア、標的型攻撃、SQLインジェクション等）
- 暗号技術（共通鍵、公開鍵、ハッシュ、デジタル署名）
- 認証技術（多要素認証、バイオメトリクス、PKI）
- ネットワークセキュリティ（FW、IDS/IPS、VPN、WAF）
- セキュリティ管理（ISMS、リスクマネジメント、インシデント対応）
- 法規（個人情報保護法、不正アクセス禁止法等）
- テクノロジ全般（ネットワーク、データベース）
- マネジメント（システム監査、PM、サービスマネジメント）
