間違えた問題の記録を分析し、弱点を特定して対策を提案してください。

## 手順

1. `progress/wrong-answers.md` を読み込む
2. 間違えた問題を分野ごとに分類・集計する
3. 弱点パターンを分析する

## レポート形式

```
## 弱点分析レポート

### 間違いの多い分野 TOP 3
1. [分野名]: X問不正解 — [共通する間違いパターン]
2. [分野名]: X問不正解 — [共通する間違いパターン]
3. [分野名]: X問不正解 — [共通する間違いパターン]

### よくある間違いパターン
- [パターン1の説明と対策]
- [パターン2の説明と対策]

### 集中復習すべきトピック
1. [トピック名] — [なぜ重要か、どう覚えるか]
2. [トピック名] — [なぜ重要か、どう覚えるか]
3. [トピック名] — [なぜ重要か、どう覚えるか]
```

4. 弱点分野から5問の復習クイズを出題する
5. 関連する概念の簡潔な再解説を行う
