7日間の学習計画に対する進捗状況を確認し、レポートしてください。

## 手順

1. `study-plan.md` を読み、全体の学習計画を把握する
2. `progress/` ディレクトリ内の学習記録を確認する
3. `progress/wrong-answers.md` から弱点分野を分析する
4. 今日の日付から試験日（2026年4月26日）までの残り日数を計算する

## レポート形式

```
## 学習進捗レポート

📅 今日: [日付] | 試験まで: あと[N]日
━━━━━━━━━━━━━━━━━━━━━━━

### 進捗状況
| 日 | テーマ | 状態 |
|---|--------|------|
| Day 1 | セキュリティ基礎 | ✅完了 / 🔄進行中 / ⬜未着手 |
| Day 2 | 暗号・認証技術 | ... |
| ... | ... | ... |

```
